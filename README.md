# Jetson Robotics Learning Roadmap

这是一个面向 **Nvidia Jetson 机器人开发岗位** 的学习计划仓库。

目标岗位方向包括：

- Nvidia Jetson 平台开发
- ROS1 / ROS2 机器人系统开发
- 常见传感器与执行器适配
- 深度学习与强化学习部署
- Isaac Gym / MuJoCo / Genesis 仿真训练
- Real2Sim / Sim2Real 项目复现
- 开源项目维护与技术文档输出

## 学习目标

通过系统学习和项目实践，最终具备以下能力：

- 能够在 Jetson 平台上配置开发环境、部署模型并进行性能优化
- 能够使用 ROS2 接入相机、雷达、IMU、电机等机器人设备
- 能够使用 C++ 和 Python 开发机器人感知、控制和部署模块
- 能够复现强化学习、模仿学习和具身智能开源项目
- 能够输出 GitHub 项目、README、技术文档和 Demo 展示
- 能够参与 GitHub、ROS、Nvidia、Discord 等开源社区交流

## 每日学习强度

计划每天学习 **10 小时**。

建议分配：

```text
2小时：理论学习
3小时：代码练习
3小时：项目实战
1小时：英文文档 / 论文 / 开源项目阅读
1小时：学习总结 / GitHub提交 / README编写
```
